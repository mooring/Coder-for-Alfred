# Coder

Speed up your normal coding transform like md5, sha1, base64 encode, base64 decode...

*and all of this is done offline, without any net requests*

![Screen Shot](https://github.com/mooring/osx-software-configs/blob/master/Alfred/screen-shoot.png?raw=true)

## features

- md5sum
- sha1
- base64 encode
- base64 decode
- url encode
- url decode
- json beautify
- json minimize
- tidy xml
- ip2long
- long2ip


## usage

### by shortcut keys

1. selecting text , and then press Ctrl + Alt + Cmd + J
2. select algorithm and then press enter key
